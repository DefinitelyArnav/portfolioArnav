Satark Crime Mapper and SIH 2018

Apr 2, 2018 2 min read R, Events & Competitions

(Disclaimer: The data used for this application is synthetic. I’ve used London city’s data and modified it by randomly attaching districts, geolocation, victim_age etc. The crime records reported in the application are in no way related to Assam.)

During the Smart India Hackathon 2018, I got a chance to make an interactive ‘Crime Mapper and Spatial data analytics tool’. My team, Six Sigma, named it ‘SATARK’.

The code is entirely written in R and I’ve used R’s ‘Shiny’ package to build it into an interactive app. For those who don’t know, building interactive apps in R with Shiny is really easy. Shiny package takes care of the front end and back end part on it’s own and thus you can focus entirely on building the analytics engine for your application. Further you can deploy and share you application for free using shinyapps.io.

I would be basing my first few tutorials upon the learnings I gathered during this amazing hackathon finale, which I was really fortunate to experience. Till then feel free to play around with ‘Satark’. Oh and the app is mobile responsive too!

Here’s the link for the same: Satark Crime Analysis

For those who would like to venture into the code on their own, here it is. Satark@definitelyArnav.Git
